# DelayBank

A multi-bank delay effect with retro styling, basic filters, and bitcrusher.

## Features
- 6 parallel delay banks
- Sync or free-time per bank
- Global high-pass and low-pass filters
- Bitcrusher with rate and depth
- Preset selector

## Build

```sh
cargo build -p delaybank
cargo xtask bundle -p delaybank
```
